# Parser Excel File
Simple WPF project that read an export excel file from Prom.ua and convert it to format that OKAYCMS can read

## How it works?
1. You open a file from Prom
2. Click on a button
3. Choose where to save parsed file(file will be saved in (.xlsx) type)
4. Open file and save as (.csv) (it's important to OKAYCMS)
